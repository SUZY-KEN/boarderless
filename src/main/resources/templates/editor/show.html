サイトの狙い

AIM
  • 初心者スケーターの練習場所探し
  • 自分の作成したスポットへスケーターを誘致
  • パークやショップの広告

上記を想定してこのサイトを作りました。皆様でいろいろな使い方をしてくれると嬉しいです。

サイトの使い方

PURPOSE

スポットの並び替え

+

表示されるスポットは評価の高い順に並んでいます。

投稿順のボタンをタップすることで直近の投稿時間のスポットを閲覧することができます。

スポットを評価の高い順に戻したいときはもう一度投稿順のボタンをタップしてください。

スポットのソート

+

カテゴリーの指定することでスポットをソートすることができます。

ロゴ横のボタンを一つ選びタップするとそのカテゴリー該当するスポットを閲覧することができます。

タップするとオレンジ色になります。

全てのスポットを表示したい場合はオレンジ色のボタンをタップしてください。

アンカー機能

+

スポットの詳細タブを開くと画像の下に案かアンカーというボタンが出てきます。

タップすると詳細タブを開いているスポットを画面の中央に表示します。

スポットの登録手順

+
  1. 左のメニューバー（スマホは上部）のスポット登録ボタンをタップ
  2. SpotRegisterの表示が画面左下（スマホは右上）に表示されたら、登録したいスポットのマップ上の位置をタップ
  3. スポット登録フォームに記入し、登録ボタンをタップ
  4. 管理者が承認するまでお待ちください
  5. 承認されたら、お知らせボックスに承認された旨のメッセージが届きます
  6. スポット一覧横のマイスポットにてスポットが登録されているか確認できます

お気に入りについて

+

スポット詳細タブのお気に入りボタンをタップするとお気に入りスポットを登録することができます。

お気に入りしたスポットはスポット一覧の右のお気に入りボタンをタップすることで、閲覧することができます。

通報機能について

+

スポット詳細タブに通報ボタンを設けております。

問題のあるスポットの何が問題かを詳細に記述していただけると幸いです。

また、問題度が高い場合にはお問い合わせよりご連絡いただけると幸いです。

登録スポットでのトラブル

+

ユーザーの方へ

スポットの承認はこちらで管理していますが、いたずらでの登録や不適切な表現があった際に弾く意図で行っていますので、

登録されているスポットでの土地所有者とのトラブルなどの現場で起こる諸々には一切責任を負いかねますのでご了承ください。

トラブルに遭われた方

土地の所有者や近隣の住民の方でトラブルに遭われた方は、サイト内の通報ボタン又はお問い合わせよりご連絡ください。

内容によってスポットの削除を行わせていただきます。

こちらではトラブルの責任は一切負いかねますのでご了承ください。

登録スポットの管理について

+

承認の作業を人力で行っていますので、スポットの登録処理をおこなってからの待ち時間にばらつきが出てくると思いますが、ご容赦ください。

被害の声が上がったスポットに関しましては、勝手ながらこちらで削除することがありますので、ご理解いただけますと幸いです。

スケートボードに関する法規

+

ユーザーが道路交通法第76条第4項第3号の「交通のひんぱんな道路において、球戯をし、ローラー・スケートをし、またはこれらに類する行為をすること」に抵触しないようにスポットの管理をしていきます。

こちらでは交通の頻繁な道路か判定しづらい場合がありますので、ユーザーの皆様の判断でスポットの申請をしていただけますと幸いです。
